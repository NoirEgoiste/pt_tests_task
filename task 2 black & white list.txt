xтестовое задание

Ограничения: 
Python 3.1 


Краткое описание: Проверяется запускаемое ПО с помощью черного или
белого списка разрешений.

Описание:
Обнаружение и проверка запускаемого ПО с отображением даты запуска. 
Запускаемое ПО не имеющее отношения к работе вашей организации может нести в
себе угрозу безопасности данным, повышает вероятность заражения ПК и
компьютерных сетей.
Для обеспечения безопасности требуется контроль за ПО запускаемом на ПК.

Информация берется из %WinDir%\Prefetch.
Контроль требует прав доступа к директории и файлам %WinDir%\Prefetch


В программе можно задавать список файлов в виде тесктового файла
"имя_файла.exe", (другие расшиения не учитывать) для контролирования запуска
ПО по белому или черному списку. Список задается в переменной "Список ПО"
Каждый софт отделен друг от друга новой строкой.

пример: 
имя_файла1.exe
имя_файла2.exe

В переменной "Тип проверки" выбирается тип сравнения
списка - Белый список или Черный список.

Белый список - только разрешеные программы
Черный список - проверяется на отсутствие запроещенных программ 

Программа выводит список ПО с датами запуска в формате:
Наименование запускаемого файла | дата запуска выделяя жирным шрифтом
(обрамляемся тегами текст <b> <\b>) ПО несоответствующее условиям контроля.

В случае отсутствия прав на получение информации от файлов программа выдает
конечный вердикт (статус) о безопасности.
Возможные статсуы: соответствует/не соответствует

Соответстие достигается:
- когда в треуемой папке находятся программы только из "белого списка"
- когда в треуемой папке отстутствуют программы из "черного списка"

Остальные случаи не относятся к соответствию требованиям. 

Результирующию таблицу выводить в отдельный файл расширения txt
Список отсортировать по алфавиту
